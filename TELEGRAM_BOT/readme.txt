## Telegram бот - AnderBot

Приветствую! Это репозиторий Telegram бота “AnderBot”, который был создан в рамках выполнения задания на создание Телеграм чат бота.

## БОТ ТЕЛЕГРАММ

CHAT_BOT_TG:@ander_tg_bot 

## О проекте

Цель проекта - разработать чат-бота для Telegram на языке Python с использованием библиотеки telebot.

## Основные функции бота

- Обработана команда «/start»
- Обработана «/help»
- echo_meddage (Ответ пользователю на сообщения его же текстом)
- Отправка случайного стикера из набора
- Генерация случайного числа от 1 до 100 и его отправка пользователю
- Отправка гс ботом
- Запрос имени пользователя и отправка персонализированного приветствия
- Предоставление меню с кнопками: “Каталога” с несколькими подразделами
- Запись каждого запустившего бот в users.db c помощью sqlite в формате (id auto_increment primaty key ,username varchar(50) ,date varchar (50) )

## Теоретические вопросы по Python для создания Telegram бота

1. Как создать бота в Telegram и получить API ключ?
- Ответ: Bot_Father — оффициальный бот Telegram, предназанченный для создания любого чат-бота. 

	1.Запускаем.
	2.Вводим название бота
	3.Вводим %tg_username% Бота
	4.Получаем ссылку на бота и его API токен.

2. Какие типы сообщений могут обрабатывать боты в Telegram?
- Ответ: В зависимости от указанного в декораторе @bot.message_handler в поле content_types, чат бот может обработать текстовые сообщения, изображения, стикеры, голосовые сообщения и другие типы медиа-сообщений.

3. Как отправить текстовое сообщение от бота в Telegram с использованием Python?
- Ответ: Используется метод send_message() чтобы просто отправить текстом или reply_to() чтобы ответить на конкретное сообщение пользователя

4. Как обработать команды от пользователя в Telegram боте на Python?
- Ответ: Для обработки команд от пользователя в Telegram боте используется декоратор
«@bot.message_handler(), внутри которого помещен»

«commands=['%название_команды%']»

5. Как отправить изображение от бота в Telegram с использованием Python?
- Ответ: Для отправки изображения нужно использовать метод send_photo, с предварительной загрузкой изображения  в проект и методом open()

2. Запустите файл main.py, чтобы запустить бота.
